  • 摘要

    以中关村地区的大学科技园为例,在广泛深入实地调研的基础上,分析我国大学科技园的最新发展动态,包括基本建设情况和发展模式等;评价了大学科技园在推动区域创新和创业中的作用,剖析了大学科技园目前面临的主要问题,对大学科技园的发展提出了针对性建议.
